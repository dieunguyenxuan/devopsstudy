## Docker có 3 thành phần chính
* Docker client
* Docker Host
* Docker Registry (Docker Hub)
## Docker image
*Liệt kê các image*   
` docker images `
## Chạy một contaniner
*Chạy một contaniner từ images*  
` docker run <image:tag> `  
*Kiểm tra các container đang chạy*  
` docker ps`
